# fridgewatchdog-open-dataset

## Data Update Schedule

The dataset is automatically refreshed **every 24 hours at midnight UTC (00:00 UTC)** via GitHub Actions.

**Timezone Note:** Please note that all timestamps within the dataset are recorded in the **UTC+1** timezone.

## Data Download

The dataset can be downloaded directly from this repository in the [`data/`](https://github.com/KamilKozera/fridgewatchdog-open-dataset/tree/main/data) folder.

The data is provided in the following formats:
- **CSV (`.csv`)** – ideal for tabular data analysis in Python (Pandas), R, or other analytical tools.
- **Excel (`.xlsx`)** – convenient for direct viewing and manual data manipulation in spreadsheet software.
- **JSON (`.json`)** – suitable for application integration and handling nested data structures.

**How to download the data:**
1. **Download the entire dataset (Recommended):** You can clone the repository using Git:
   ```bash
   git clone https://github.com/KamilKozera/fridgewatchdog-open-dataset.git
   ```
   Alternatively, you can download everything as a ZIP archive by clicking the green **"Code"** button in the top right corner and selecting **"Download ZIP"**.
2. **Download individual files:** Navigate to the `data/` folder, click on the file you need, and then click the **"Download raw file"** icon in the top right corner of the file preview.
